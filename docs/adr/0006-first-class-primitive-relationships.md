# First-class primitive relationships

Primitive Relationships are first-class authored objects rather than being hidden inside each Visual Primitive as `dependsOn` or `motion`. Visual Primitives name the subject-specific objects in a diagram; Primitive Relationships describe how those objects affect, support, intersect, feed, reveal, or transform each other through a relation, visual metaphor, motion, and timing role. This was chosen because prompt-shaped videos need to draw mechanisms and choreography, not just place labeled objects on a canvas.
