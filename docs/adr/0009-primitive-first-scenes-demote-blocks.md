# Primitive-first scenes demote blocks

Status: Superseded by ADR 0012

Non-graph scenes are primitive-first: Visual Primitives and Primitive Relationships carry the explanation, while blocks become supporting captions or callouts. Blocks remain useful for concise context, but layout should preserve the primitive diagram before block descriptions. This was chosen over making primitives the only content because captions help comprehension, and over keeping card blocks primary because that would keep physical/scientific/built-world prompts visually dominated by the same generic card layout.
